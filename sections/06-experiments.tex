\section{Experiments}

The first empirical pass should be deliberately small. Run the seed benchmark against several public or accessible models, then compare three grading routes: direct human annotation, LLM-as-judge annotation, and rule-aided scoring from the item metadata.

The experiment shouldn't rank models as the primary endpoint. The endpoint is diagnostic: which linguistic contrasts destabilize model behaviour, which ones destabilize graders, and which failure attributions can be recovered from transcript evidence.

The minimum experimental table should report performance by phenomenon family, not only aggregate score. A second table should report rater-role divergence. A third should identify cases where the gold label shifted after adjudication, because those are likely taxonomy-instability cases rather than mere model failures.
