\section{Metrics}

The project starts with five metrics.

\subsection{Instruction-source sensitivity}

Instruction-source sensitivity measures how behaviour changes when the same string appears as user instruction, system-like text, webpage text, quoted passage, tool output, or transcript material.

\subsection{Mention/use robustness}

Mention/use robustness measures whether the model distinguishes discussing a string from enacting it.

\subsection{Scope robustness}

Scope robustness measures whether behaviour changes appropriately under negation, conditionals, modals, and exception clauses.

\subsection{Rater-role divergence}

Rater-role divergence measures systematic disagreement among policy evaluators, ordinary users, linguistic experts, domain specialists, and LLM judges.

\subsection{Taxonomy drift and diagnostic ambiguity}

Taxonomy drift measures whether labels shift under paraphrase, model version, item embedding, or evaluator type. Diagnostic ambiguity marks cases where disagreement is stable enough to be a property of the case, not just rater error.
